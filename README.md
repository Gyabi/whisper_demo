# whisper_demo
OpenAIによって公開されているWhisperをDockerより駆動させるデモリポジトリ

# devices
- Windows11
- WSL2
- docker
- nvidia-docker2
- RTX 3060


# how to
本リポジトリのクローン
> a

参照するDockerImageをローカルに落とす。
> docker pull nvcr.io/nvidia/pytorch:22.09-py3

# reference
[こちらのサイト](http://pro-video.jp/voice/announce/)よりデモ音声を利用しています。
